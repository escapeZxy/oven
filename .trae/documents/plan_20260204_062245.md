# 重新设计训练计划列表页 UI

为了提升用户体验和视觉吸引力，我计划将当前的 Material 列表风格升级为更现代、更具“控制面板”感的 UI。

## 设计目标
1.  **强化当前计划**：使用 Hero Card 风格突出显示用户正在进行的计划，并提供明确的进度反馈。
2.  **现代化布局**：将计划库从简单的列表改为响应式网格卡片，提高信息密度和可读性。
3.  **增强交互感**：添加悬停效果、图标和更好的色彩层次。
4.  **清晰的层次结构**：通过排版和间距区分不同功能区域。

## 实施方案

### 1. 结构重组
- **顶部标题栏**：添加一个清晰的标题和“创建新计划”按钮，使用更现代的按钮样式。
- **当前计划 (Hero Section)**：
    - 使用深色渐变背景（Indigo/Purple）和圆角设计。
    - 左侧展示计划名称、描述和当前训练日名称。
    - 右侧或下方展示一个进度条，显示已完成的百分比。
    - 提供一个巨大的、醒目的“开始今日锻炼”按钮。
- **计划库 (Grid Section)**：
    - 采用三列网格布局。
    - 每个卡片包含图标、标题、简短描述和训练天数标签。
    - 移除原本简单的 `mat-list`，改用自定义设计的卡片。

### 2. 技术实现
- **Angular 组件**：在 `LogListComponent` 中导入 `MatProgressBarModule` 以支持进度显示。
- **Tailwind CSS**：利用 Tailwind 的实用类处理布局、间距、颜色和悬停动画，减少对自定义 CSS 的依赖。
- **清理**：移除 `log-list.component.css` 中不再使用的旧样式代码。
